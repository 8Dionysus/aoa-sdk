# Via Negativa

This part owns the SDK pruning checklist for antifragility-shaped helper
surface pressure.

## Role

Input: proposed helper surface, duplicate owner operation, stale policy cue,
alternate-path pressure, or mutation wrapper.

Output: keep, merge, move, suppress, quarantine, deprecate, or remove route
with owner and provenance preserved.

Owner: `aoa-sdk` owns the checklist for SDK helper surfaces. Owner
repositories decide owner-local deletion, replacement, repair, proof, and
runtime behavior.

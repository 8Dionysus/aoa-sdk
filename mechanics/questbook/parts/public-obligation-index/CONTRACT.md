# Public Obligation Index Contract

## Contract

`QUESTBOOK.md` is a readable index over durable SDK obligations.

## Must

- point back to source records under `quests/`;
- distinguish open obligation visibility from roadmap direction;
- keep owner boundaries explicit.

## Must Not

- author quest source truth without a matching source record;
- use the index as release readiness, proof verdict, or owner acceptance;
- turn generated readers into source truth.

# Titan Parts

Functioning Titan SDK parts live here. Each active part provides a local
operation card, contract, validation route, and artifact homes.

## Active Parts

- [incarnation-identity-runtime-helper-contracts](incarnation-identity-runtime-helper-contracts/README.md) -
  keeps runtime receipt, incarnation spine, identity ledger, ingress, gates,
  and titanctl helper contracts bounded.
- [operator-console-helper-contracts](operator-console-helper-contracts/README.md) -
  keeps console roster, event, gate, approval, digest, and appserver-plan
  helper contracts bounded.
- [appserver-bridge-helper-contracts](appserver-bridge-helper-contracts/README.md) -
  keeps app-server bridge session, event replay, launch-plan, approval, and
  metrics helper contracts bounded.
- [memory-loom-recall-helper-contracts](memory-loom-recall-helper-contracts/README.md) -
  keeps memory loom, recall, retention, and redaction helper contracts bounded.
- [session-praxis-replay-helper-contracts](session-praxis-replay-helper-contracts/README.md) -
  keeps visible-session replay, phase graph, packets, compaction source, and
  learning-delta helper contracts bounded.
- [swarm-ledger-closeout-helper-contracts](swarm-ledger-closeout-helper-contracts/README.md) -
  keeps task, report, finding, grade, timeout, ledger, and closeout helper
  contracts bounded.

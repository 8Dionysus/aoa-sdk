# Hook Observation Pack

## Use When

Use this part when recurrence hook binding sets, hook producers, or hook run
reports change.

## Do Not Use When

Do not use it as an orchestration, repair, or scheduler surface.

## Route Check

Input is a hook binding set plus an event. Output is an observation-shaped hook
run report.

## Active Outputs

- hook binding set schema and examples
- hook run report schema and examples
- hook producer guide
- technique publication observation boundary

# Recursor No-Spawn Law

This recurrence route prepares future recursor agents but does not run them.

Spawn, run, summon, and open-arena subcommands are outside this recurrence
route.

Any future command that resembles these verbs must fail owner review unless a
later explicit agent/runtime route authorizes it.

The current SDK seam is limited to readiness, boundary-check, and
projection-candidate inspection.

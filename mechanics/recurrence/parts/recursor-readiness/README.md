# Recursor Readiness

## Use When

Use this part when SDK recursor readiness scanners, no-spawn law, or projection
candidate checks change.

## Do Not Use When

Do not use it to install Codex agents, spawn agents, open arenas, or activate a
recursor pair.

## Route Check

Input is `aoa-agents` recursor readiness configuration. Output is a read-only
readiness, boundary, or projection-candidate report.

## Active Outputs

- recursor readiness docs
- readiness and boundary examples
- readiness schemas
- read-only scanner script

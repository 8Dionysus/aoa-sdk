# Review Decision Closure

## Use When

Use this part when an owner review queue needs a decision template, close report,
decision ledger, or suppression memory.

## Do Not Use When

Do not use it to decide on behalf of the owner.

## Route Check

Input is a review queue plus explicit owner decisions. Output is a closure
packet that preserves unresolved work.

## Active Outputs

- owner review decision schema and examples
- review decision close report schema and example
- decision ledger and suppression memory schemas

## Next Route

Owner repos perform any accepted follow-through; downstream projections may
consume the closure as advisory evidence.

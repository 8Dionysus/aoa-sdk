# Downstream Projection Guard

## Use When

Use this part when recurrence emits routing, stats, or KAG projection packets.

## Do Not Use When

Do not use it to publish full graph truth or transfer owner authority to a
projection consumer.

## Route Check

Input is reviewed recurrence evidence. Output is a thin downstream projection
plus a guard report.

## Active Outputs

- routing projection schema and example
- stats projection schema and example
- KAG projection schema and example
- downstream bundle and guard report schemas

## Next Route

Projection consumers inspect advisory packets and keep their own owner truth.

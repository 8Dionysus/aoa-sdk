# Live Observation Producers

## Use When

Use this part when recurrence needs to collect live observations from existing
owner-authored surfaces.

## Do Not Use When

Do not use it to schedule work, spawn agents, promote candidates, or mutate
owner repos.

## Route Check

Input is an owner surface plus a producer filter. Output is an observation
packet.

## Active Outputs

- live observation command
- live observation run schema and example
- producer list tests

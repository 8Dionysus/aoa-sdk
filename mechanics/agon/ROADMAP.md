# Agon Roadmap

This roadmap owns future pressure for the SDK Agon mechanic.

## Current Contour

Agon is landed as candidate-only SDK helper pressure: center-law previews,
state-packet review bindings, recurrence adapters, duel-kernel bindings,
verdict/retention/rank helper candidates, epistemic/KAG helper candidates,
school-lineage campaign helpers, and Sophian-threshold helpers.

The mechanic helps inspect and prepare owner-review surfaces. It does not run
Agon, mint verdicts, mutate rank, write scars, or promote anything to
Tree-of-Sophia.

## Next Work

- Keep every generated helper candidate tied to a source owner and an explicit
  stop-line set.
- Keep recurrence prebinding lanes optional, review-only, and visible as
  candidate support rather than scheduler behavior.
- Add Agon helper depth only when the owner surface, review mode, generated
  builder, and validator are all clear.

## When Time Comes

- Add a new Agon part only when the pressure cannot stay inside an existing
  helper family without making that family less readable.
- Promote a helper closer to execution only after the Agon owner names the
  executable meaning and rollback posture.

## Out Of Scope

- live arena sessions;
- verdict, rank, retention, scar, or ToS authority;
- hidden recurrence scheduling;
- proof or memory acceptance.

## Update Trigger

Update this roadmap when Agon helper scope, review posture, generated helper
families, or future promotion conditions change.

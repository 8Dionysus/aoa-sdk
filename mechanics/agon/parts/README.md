# Agon Parts

Functioning Agon SDK parts live here. Each active part provides a local
operation card, contract, validation route, and artifact homes.

## Active Parts

- [center-law-preview-helpers](center-law-preview-helpers/README.md) - previews
  center contradiction, closure, and summon law as candidate-only SDK helpers.
- [state-packet-review-bindings](state-packet-review-bindings/README.md) -
  binds packet, sealed-commit, reveal, and packet stop-line helpers for review.
- [recurrence-adapter](recurrence-adapter/README.md) - adapts Agon recurrence
  review lanes into candidate-only SDK registries without becoming Agon
  runtime or verdict authority.
- [duel-kernel-review-bindings](duel-kernel-review-bindings/README.md) -
  prepares duel kernel and mechanical-trial SDK bindings for owner review.
- [verdict-retention-rank-review-helpers](verdict-retention-rank-review-helpers/README.md) -
  keeps VDS, retention, rank, and review-bundle helpers candidate-only.
- [epistemic-kag-review-helpers](epistemic-kag-review-helpers/README.md) -
  prepares epistemic and KAG review helpers without canon or promotion
  authority.
- [school-lineage-campaign-review-helpers](school-lineage-campaign-review-helpers/README.md) -
  keeps SLC helper contracts non-authoritative.
- [sophian-threshold-review-helpers](sophian-threshold-review-helpers/README.md) -
  prepares Sophian threshold review helpers without ToS canon writes.

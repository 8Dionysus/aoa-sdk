# Portability Boundary

## Role

`portability-boundary` explains how SDK workspace discovery relates to live
Codex deployment when the AoA workspace root changes.

## Input

- workspace layout docs
- SDK workspace discovery behavior
- source-owned 8Dionysus regeneration route

## Output

- `docs/portability-boundary.md`

## Owner

`aoa-sdk` owns the boundary explanation. 8Dionysus and host/Codex deployment
owners retain deploy regeneration and runtime authority.

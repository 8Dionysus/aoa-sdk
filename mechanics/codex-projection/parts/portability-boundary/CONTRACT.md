# Portability Boundary Contract

## Contract

This part may tell operators when SDK discovery overrides are appropriate. It
must route deploy regeneration back to source-owned 8Dionysus Codex projection
surfaces.

## Stop-Lines

- Do not patch SDK code or MCP server names to hide deployment-path drift.
- Do not make SDK workspace discovery stronger than the source-owned deploy
  renderer.
- Do not claim ownership over `8Dionysus/.codex/config.toml` or
  `8Dionysus/.codex/hooks.json`.

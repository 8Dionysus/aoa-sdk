# Agon Recurrence Review Lanes

The current recurrence review schema already supports these lanes:

```text
technique
skill
eval
playbook
general
```

The adapter uses:

```text
general    center, agents, routing
playbook   trial playbooks
technique  technique candidates
skill      skill candidates
```

Future Wave VII may extend or refine lanes for eval, memo, and stats. This adapter does not require that extension.

# aoa-sdk Skill Home

This is the canonical home for callable procedures whose executable helper
shape is owned by `aoa-sdk`.

The intended Titan family has three front doors:

| Bundle | Internal modes | Stronger owner stop-lines |
|---|---|---|
| `titan-console` | console, approval witness, approval queue, plan, receipt | operator consent and live runtime |
| `titan-appserver-bridge` | bridge, event replay | Codex/app-server runtime and hidden session truth |
| `titan-memory-loom` | ingest, recall, retention handoff | durable memory and retention acceptance in `aoa-memo` |

The semantic graph and KAG index are derived consumers. The admitted v2 owner
port exposes these sources to the single managed OS user profile rather than
creating a repository-local duplicate.

Unavailable runtime guards remain graph capabilities with unavailable
bindings, not fictional skills. Titan summon stays with `aoa-agents`, and
Titan closeout remains a blocked playbook route until `aoa-playbooks-mcp`
exists.
